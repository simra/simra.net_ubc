Estimating Error

We are seeking in this section a method for combining individual estimates obtained from different tracked landmarks. This can be accomplished using the merging operation defined above if we can obtain an error model for estimates obtained from each tracked landmark. An error model for a particular tracked landmark T can be constructed using cross-validation. That is, we measure how well each observed candidate landmark in T is predicted by the rest of the candidate landmarks in T. This is a quantity which is fixed for a given tracked landmark, and hence can be computed a priori.
